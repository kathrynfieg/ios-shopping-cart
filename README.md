# ios-shopping-cart

A simlpe shopping list application developed in Swift with SQLite to store and access data locally. 
<br><br>

Features:
<ul>
  <li>Add items to list</li>
  <li>Remove items from list</li>
  <li>Change system font colour</li>
</ul>
